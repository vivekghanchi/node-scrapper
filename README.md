# Node-scrapper
Medium blog Url scrapper in Node

## Clone git repository

```bash
git clone https://github.com/vivekghanchi/node-scrapper.git
```

## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) to install all dependency.

```bash
npm install
```

## Start Server

Run app
```bash
node app
```

## Usages

```bash
http://localhost:3000/
```
